# SQL Reporting & Business Analytics Portfolio

This repository contains SQL scripts developed for finance, service, inventory, equipment, and billing analytics using ERP production data.

## Summary

These scripts support reporting, reconciliation, KPI tracking, and data-quality analysis across key business areas such as AR aging, general ledger, maintenance contracts, technician billing, fleet invoicing, equipment book value, obsolete inventory, and roll-rate analysis.

## Significance

The SQL logic helps convert raw ERP transactions into actionable business insights for Power BI dashboards, SSRS reports, financial validation, operational audits, and management decision-making.

## Key Areas Covered

- Accounts Receivable aging and roll-rate tracking
- General Ledger and account schedule validation
- Maintenance contract profitability and KPI analysis
- Technician labor rounding and billing impact analysis
- Fleet invoice and overtime billing review
- Equipment book value calculation
- Obsolete parts/write-off analysis
- Dashboard fixes and financial data reconciliation

## Business Value

These queries improve reporting accuracy, identify revenue leakage, support audit readiness, validate financial metrics, and enable faster decision-making through structured analytics-ready datasets.
